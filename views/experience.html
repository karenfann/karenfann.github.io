Response Two
Software Developer Intern
In the summer of 2017, I began working at Response Two, a start-up document analysis platform that allows anyone to create their own, private search-engine on any set of scanned documents.
The next twelve weeks proved to be an invaluable experience. As an intern at a start-up, I held a tremendous amount of responsibility over the app and developed skills across the spectrum.
I built a cross-platform desktop app using Electron, created request handlers with Node.js, wrote MySQL queries to update user information, implemented various UI changes including a dynamic file list and document viewer using Angular.js and plenty more.
Bruin Meet
Front End Developer and Designer
In April 2017, I joined a team of other developers and together we set out to create a dating app specifically for UCLA students. After four intense weeks of hacking and coding, we launched Bruin Meet as UCLA's exclusive matchmaking app and it has been live ever since.
As front end developer and designer, I am in charge of designing and developing the UI/UX of Bruin Meet. Much of what I do involves working heavily with HTML/CSS and jQuery for design and dynamics, Bootstrap and media queries for responsiveness and Handlebars for back end integration.
