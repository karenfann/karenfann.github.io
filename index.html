Projects

BoomClap

Using data generated by actual flight tests conducted at NASA research centers and noise laboratories, my team and I developed an educational GUI application called BoomClap to construct a visualization of low boom as it compares to sonic boom. I programmed an Intel Edison board using Java to activate light indicators for various levels of boom.

Autonomous Electric Vehicle

Throughout the course of my senior year, I designed and constructed an automatic electric vehicle. Using an Arduino and C++, I implemented and applied deceleration and acceleration profiles to an engine system composed of an electronic speed controller (ESC), RC motor, and bipolar stepper motor.

Pitch Perfect

Pitch Perfect is an iOS app that allows users to record themselves and play the audio back with various effects: slow, fast, high pitch, low pitch, echo, and reverb. In the development of this app, I learned Swift on XCode, navigation controlling, IBOutlets, and IBActions.

Bugs!

Bugs! is a graphical ant simulation and programming competition platform. In the simulation, colonies give birth to ants who wander around, forage for food, battle with competitor ants and grasshoppers, emit pheromones for navigation and avoid deadly poison. I developed this simulation as a course project using objected-oriented programming and polymorphism.

Watch the Bugs! simulation.

BruinNav

BruinNav is a turn-by-turn navigation system I developed as a course project. Using geospatial data indexed from Open Street Maps project database, I created a binary search tree map and implemented the A* algorithm to find and reconstruct the most optimal path.
